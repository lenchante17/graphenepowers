# Feature Windowed Example

This example shows how the updated canonical workflow can look for a `Feature` that uses:

- a macro graph
- committed windows
- graph summary plus kanban tracking
- result packets

## Files

- `execution-plan.md` — example graph and current-window planning document
- `plan-progress.md` — example machine record plus rendered views
- `result-packet-W1.md` — example window-close summary
